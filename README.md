# Lumière — Med Spa Software Landing Page

A single-file marketing/sales website for a white-labeled med spa growth platform (placeholder brand: **Lumière**).

## Live site
Once GitHub Pages is enabled, the site will be live at:
`https://<your-username>.github.io/<repo-name>/`

## Files
- `index.html` — the entire website (self-contained: HTML, CSS, fonts via CDN). Open it in any browser to preview.

## How to customize
- **Brand name:** find/replace `Lumière` / `Lumi<span>è</span>re`.
- **Colors:** edit the CSS variables at the top of `index.html` (`--plum`, `--rose`, `--gold`, etc.).
- **Booking links:** point the "Book a demo" buttons (`href="#demo"` / the final CTA) at your real booking URL.
- **Phone / testimonials / pricing:** replace placeholder values throughout.

## Publishing to GitHub Pages
1. Create a new repository on GitHub.
2. Upload `index.html` (and this README) to the repo.
3. Go to **Settings → Pages**, set source to the `main` branch, root folder, and save.
4. Wait ~1 minute; your site goes live at the URL above.
